======================
Events Routes:
======================

1) GET all Events
   URL    : http://localhost:8080/app/get-events-data
   Method : GET
   Desc   : Fetches all event records

2) POST Create Event
   URL    : http://localhost:8080/app/upload-events-data
   Method : POST
   Desc   : Creates a new event
   Payload (application/json):
   {
     "category": "Wedding",
     "title": "Beachside Wedding",
     "description": "Luxury beach wedding package",
     "location": "Goa",
     "price": 50000,
     "service": "Catering, Photography",
     "thumbnail": "https://example.com/thumb.jpg",
     "banner": "https://example.com/banner.jpg",
     "videoURL": "https://example.com/video.mp4",
     "gallery": ["https://example.com/img1.jpg", "https://example.com/img2.jpg"],
     "reviews": ["Amazing", "Beautiful arrangements"],
     "date": "2025-08-20T18:30:00.000Z"
   }

3) DELETE Event
   URL    : http://localhost:8080/app/delete-events-data/:id
   Method : DELETE
   Desc   : Deletes an event by ID

4) GET Filters
   URL    : http://localhost:8080/app/get-events-filter-data
   Method : GET
   Desc   : Fetch distinct categories, locations, and price range

5) GET Events by Category
   URL    : http://localhost:8080/app/get-events-by-category/:category
   Method : GET
   Desc   : Fetch events based on category
